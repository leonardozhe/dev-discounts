## 产品定位

Token Plan 是阿里云百炼平台推出的**大模型通用节省计划**，核心解决一个问题：

> 企业使用多个大模型时，如何简化计费、降低成本？

传统模式下，每个模型单独计费，管理复杂。Token Plan 将额度统一，一个账户覆盖所有模型。


## 核心技术能力

### 1. 统一 API 接口

```python
# 兼容 OpenAI SDK
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_DASHSCOPE_KEY",
    base_url="https://dashscope.aliyuncs.com/compatible-mode/v1"
)

# 调用任意模型
response = client.chat.completions.create(
    model="qwen-max",  # 或 qwen-plus, deepseek-r1, etc.
    messages=[
        {"role": "user", "content": "分析这段代码的性能问题"}
    ]
)
```
