## 技术亮点

### 1. 万级文件精准生成

传统 AI 编程工具只能理解当前文件或相邻文件，Qoder 可以：

- 索引整个代码库 (万级文件)
- 理解模块间依赖关系
- 生成符合项目架构的代码

### 2. AI 增强检索

不只是关键词搜索，而是语义理解：

```
查询: "用户认证的逻辑在哪里？"
结果: 定位到 auth/ 目录下的相关文件，并展示调用链
```

### 3. 异步委派

复杂任务无需等待：

```
开发者: "重构这个模块，使用新的设计模式"
Qoder: 异步执行，完成后通知
开发者: 继续其他工作，稍后 review 结果
```
